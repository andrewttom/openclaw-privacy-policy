Privacy Policy — Tom Family Assistant (OpenClaw)

Last Updated: March 3, 2026

Summary

Tom Family Assistant is a private, personal productivity assistant used only by Andrew Tom. It reads and sends email and accesses calendar events only at the user’s request. Data is not shared or sold.

Data Collected

  • Email content and metadata (subject, sender, date)
  • Calendar events (title, time, attendees)
  • User instructions and preferences

How Data Is Used

  • Generate summaries and prep notes
  • Draft and send emails when explicitly requested
  • Create or edit calendar events when explicitly requested

Data Storage

Data is stored locally on the user’s OpenClaw environment. No third‑party storage or advertising platforms are used.

Data Sharing

No data is sold or shared with third parties.
